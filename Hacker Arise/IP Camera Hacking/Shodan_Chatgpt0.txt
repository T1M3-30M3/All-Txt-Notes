🔍 Core Shodan Filters (Learn These First):
org:"Company Name" → filter by organization
product:"Software/Device" → specific product
country:"BD" → country (Bangladesh example)
port:80 → specific port
hostname:"example.com" → domain-related
os:"Windows/Linux" → OS detection
title:"login" → page title
http.html:"keyword" → keyword in HTML
ssl:"keyword" → SSL cert info
net:"IP/CIDR" → IP range
