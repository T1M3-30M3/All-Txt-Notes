Google Dork - API Endpoints ⚙️
site:example[.]com inurl:api | site:*/rest | site:*/v1 | site:*/v2 | site:*/v3

Find juicy API Endpoints for further testing 🎯

site:example[.]com - Put your bug bounty target here!
inurl:api - Finds URLs with "api".
site:*/rest - Targets RESTful APIs.
site:/v1 | site:/v2 | site:*/v3 - Looks for API version paths

I used "site:" instead of "inurl:" because it removes false positives due to the way google queries work.
