========================================

- Check if URL is guessable/bruteforceable

- Submit invalid creds + valid captcha
- Capture request and replay multiple times

- Inject random IP in X-Forwarded-For header

[Missing Password Confirmation]
- Change email
- Change password
- Delete account

[Missing Email Verification]
- Register account without verifying email

[Mail Bombing]
- Capture request → Intruder → send multiple times

- Repeat form/login requests 20–30 times
- Observe no restriction

[Weak Login Protection]
- Bruteforce passwords using wordlists

[Username Enumeration]
- Compare error messages for valid/invalid users

- Check default creds from public databases

- Secret not rotatable

- No failsafe

  numeric only
  lowercase only
  common passwords
  short passwords

[Weak HTTP Usage]
