1. Detect Windows Executable Files

rule Detect_PE_File {
    meta:
        description = "Detects Windows executable files (PE)"
    strings:
        $mz_header = { 4D 5A } // "MZ" header
    condition:
        $mz_header at 0
}

  • What It Does: Detects .exe files by matching the "MZ" header at the start of the file.

2. Look for Suspicious Strings

rule Detect_Suspicious_Strings {
    meta:
        description = "Detects files with suspicious strings"
    strings:
        $cmd1 = "cmd.exe"
        $cmd2 = "powershell.exe"
        $url1 = "http"
        $url2 = ".onion"
    condition:
        any of ($cmd*) or any of ($url*)
}

  • What It Does: Flags files containing common suspicious strings like cmd.exe, powershell.exe, or URLs.

3. Detect Packed or Compressed Files

rule Detect_Packed_Files {
    meta:
        description = "Detects packed or compressed executable files"
    condition:
        uint16(0) == 0x5A4D and filesize < 500KB
}

  • What It Does: Identifies small .exe files, often packed or compressed by malware to evade detection.

4. Check for Dangerous Functions

rule Detect_Dangerous_Functions {
    meta:
        description = "Detects dangerous API calls used by malware"
    strings:
        $func1 = "VirtualAlloc"    // Allocates memory
        $func2 = "CreateProcess"  // Creates new process
        $func3 = "GetProcAddress" // Gets function from DLL
    condition:
        any of ($func*)
}

  • What It Does: Detects files containing function names commonly abused by malware.

5. Detect XOR-Encoded Data

rule Detect_XOR_Encoded_Data {
    meta:
        description = "Detects files with XOR-encoded data"
    strings:
        $xor_pattern = { 33 ?? 8B ?? } // Common XOR pattern
    condition:
        $xor_pattern
}

  • What It Does: Matches files containing patterns indicative of XOR encoding, often used to obfuscate malicious data.

6. Identify Hidden IPs

rule Detect_IP_Addresses {
    meta:
        description = "Detects embedded IP addresses in files"
    strings:
        $ip_pattern = /[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}/
    condition:
        $ip_pattern
}

  • What It Does: Matches any file containing an IP address pattern, including C2 servers.
