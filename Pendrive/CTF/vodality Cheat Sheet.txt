volshell: Starts an interactive shell where you can use Volatility commands.
volshell -h: Displays help information for the volshell command.
volshell -p <PID>: Opens a memory dump and automatically sets the context to the specified process ID.
volshell -f <MEMORY_DUMP>: Opens a specific memory dump file.
imageinfo: Retrieves basic information about the memory dump, such as the operating system, architecture, and time of acquisition.
windows.pslist: Lists the running processes.
windows.pstree: Displays a process tree.
windows.psxview: Lists hidden processes.
windows.netscan: Scans for network-related artifacts.
windows.net: Lists network connections and related information.
windows.netstat: Shows network connections.
windows.sockets: Lists network sockets.
windows.registry.hivelist: Lists the active registry hives.
windows.registry.printkey: Prints the contents of a registry key.
windows.registry.find: Searches for specific values or keys within the registry.
windows.registry.apihooks: Lists registry API hooks.
windows.registry.userassist: Lists UserAssist information from the registry.
windows.registry.shellbags: Lists shellbags information from the registry.
windows.registry.userassist: Lists UserAssist information from the registry.
windows.registry.userassist.history: Lists UserAssist history.
linux.pslist: Lists processes in a Linux memory dump.
linux.pstree: Displays a process tree in Linux.
macosx.lsadump: Dumps the LSASS process memory on macOS.
