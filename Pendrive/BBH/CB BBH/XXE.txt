XXE:

XXE stands for "XMS External Entity." It is an application layer cyber security attack that exploits an XXE vulnerability to parse XML input. It is basically used to read sensitive files such as id_rsa, config.php, etc. This can damage organizations in various ways, including DOS, sensitive data exposure, SSRF, and port scanning from the processor's locations.
