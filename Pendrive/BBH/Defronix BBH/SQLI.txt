Union Based - Get Based:
01. use \ and try to use balancer
02. use +order+by+ for finding how many column
03. use +union+all+select for finding vulnerable table number 
04. for find database use database() and find user() where is vulnerable table number
05. for find full database use "Union Base > Database > Database Name group contact"
06. for find full tables use "Union base > table > Table Name group contact"
07. For find full Columns use "Union base > columns > Column Name group Concat"
08. For find full data use "Union base > data > data group contact"

Post Based:
