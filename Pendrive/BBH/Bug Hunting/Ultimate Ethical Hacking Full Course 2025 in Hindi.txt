* Different Type Of Hackers : 
-> White Hat Hackers
-> Black Hat Hackers
-> Grey Hat Hackers
-> Script Kiddies
-> Hacktivists
-> State-Sponsored Hackers

* Testing Approaching
-> White Box Testing == Full Knowledge of Code == Internal Structure and Logic == Using and Integrating testing
-> Black Box Testing == No Knowledge of Code == Functionality and User experience == System and acceptance Testing
-> Grey Box Testing == Partial Knowledge == Both Internal and external Aspects == Integration and System testing

* 3 Team:
-> Red Team == Offensive Security == Simulating attacks, Penetration Testing == Identifing Vulnerabilities
-> Blue Team == Defensive Security == Monitor Systems, Incident Response == Protect against Attacks
-> Purple Team == Collaboration between Red and Blue == Join exercises, knowledge sharing == Strengthen Overall security Posture

* Ethical Hacking Methodolgy: Recon/FootPrinting -> Scanning -> Ganing Access -> Maintaining Access -> Cleaning Tracks

Recon == Gathering Information about the target == dorking, whois, DNS Enum, Social Eng
Scanning == Using tools identify bulnerabilities in the target == Network mapping, Port Scanning, Vulnerability Scanning
Gaining Access == Attemping Exploit identified Vulnerabilities == SQLI, Buffer Overflow, Phishing
Maintaining Access == Establishing a persistent connection == Installing rootkits/Backdoor, Creating hidden user accounts
Clearing Records == Removing Traces of the ethical hackers activities == Modifying logs files, Deleting evidence etc

Recon -> Weaponization -> Delivery -> Exxploitation -> Installation -> Command and Control -> Action and Objective
