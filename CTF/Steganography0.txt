Steganography Tools:

stegcracker : Python-based brute-force tool for steghide.
stegseek : Fastest steghide cracker using optimized algorithms.
stegsolve : Java-based GUI tool for analyzing steganographic images.
stegdetect : Detects steganography in JPEG images.
stegbreak : Cracks jphide-based steganography.
stegosuite : GUI-based tool for hiding and extracting data in images.
stegsnow : Hides messages in whitespace (SnowSteg).
stegano : Python library for steganography operations.
stegify : Tool for hiding files inside PNG images.
steg : CLI tool for steganography in images.
steganabara : Steganography analysis and detection tool

For WAV files → stegcracker, stegseek, stegdetect
For images → stegsolve, stegosuite, stegify
For text-based hiding → stegsnow
For brute-force attacks → stegseek, stegbreak

For steganography password cracking → stegcracker, stegseek, stegbreak
For ZIP/RAR cracking → fcrackzip, rarcrack, john
For general CTF password cracking → hashcat, hydra, medusa

Steganography Password Cracking Tools:

stegcracker : Brute-force steghide-protected files.
