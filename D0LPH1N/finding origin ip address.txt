Finding Origin IP Address:
Process : ping <link> → nslookup <link> → https://securitytrails.com/ → https://search.censys.io/ → https://www.shodan.io/dashboard

* Create a account https://securitytrails.com/ or https://search.censys.io/ or https://www.shodan.io/dashboard search by domain you may find real ip

set CENSYS_SEARCHTYPE ipv4
set CENSYS_SECRET <secret key from website>
set CENSYS_UID <secret uid from website>
run
