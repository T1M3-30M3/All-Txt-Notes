###Create, Copy, Move, and Delete Commands 
cd/ [for go to drive]
cd users [for go to next step]
cd usersname
cd desktop
cd.. [for reverse 1 step]
mkdir name [for make a folder and name is your folder name]
rmdir name [for delete a folder and name is your folder name]
ren previousname nextname [for rename your folder]
cd [use for goto next]
type nul>name.extention [for make any kind of file]
copy filename destinationfile/foldername [for copy file]
move filename destinationfile/foldername [for move file]
del filename [for delete file]
del foldername[for delete all file in folder]
cls[for clear]
exit[for exit]
